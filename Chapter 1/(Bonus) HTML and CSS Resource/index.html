 HTML Tags:

  1. <b> </b>

    2. Certain words or sentences need to be emphasised, to let readers know of its importance. This tag is used to bolden texts.

    Example: 
     In this sentence, <b>this text is emphasised.</b>
     
    In this sentence, this text is emphasised.

  2. <u> </u>

    3. Certain words or sentences need to be underlined, to let readers know of its importance. This tag is used to add underlines to texts.

    Example: 
     In this sentence, <u>this text is underlined.</u>
     
    In this sentence, this text is underlined.

  3. <i> </i>

    4. Certain words or sentences need to be emphasised, to let readers know of its importance. This tag is used to add tilted texts called italics.

    Example: 
     In this sentence, <i>this text is italicised.</i>
     
    In this sentence, this text is italicised.

  4. <a> </a>

    5. This tag can be used to make links to web pages, which are clickable by the user. The attribute href is used as the website link while target is to redirect to a new website tab.

    Example: 
     Google link: <a href="https://www.google.com/" target=blank_">Google</a>
     
    Google link: Google

  5. <p> </p>

    6. this tag is used to create paragraphs of texts.

    Example: 
     <p>These are texts.</p>
     
    These are texts.

  6. <img>

    7. this tag is used to add images to the website, the image link address taken from the web, or the image's path from the user files is needed to add the image.

    Example: 
     <img src="images/apple.jpg">
     


  7. <br> </br>

    8. this tag is used to add line breaks in your texts. This can be used to add spaced out texts

    Example: 
     <br>This text.</br> That text.
     

    This text.
    That text.

  8. <title> </title>

    9. this tag is used to add a title to your website, which can be seen on the title bar of the website tab.

    Example: 
     <title>HTML and CSS Resources</title>
     
     

  9. <h1> </h1>

    10. this tag is used to add headings to a website. There are a total of 6 headings with different sizes

    Example: 
     <h1>Introduction to HTML</h1>
     

    Introduction to HTML

  10. <body> </body>

    11. this tag is used to contain the websites text elements, the elements are displayed on the website page.

    Example: 
     <body><p>This is a part of the body of the website page.</p></body>
     

    This is a part of the body of the website page.

 

 CSS Properties:

  1. color

    2. this property is used to change the colour of a text.

    Example: 
     <p style="color: green">The text's colour is green.</p>
     
    The text's colour is green.

  2. font-size

    3. this property is used to change the font size of a text.

    Example: 
     <p style="font-size: 30px">This text is big.</p>
     
    This text is big.

  3. font-style

    4. this property is used to change the font style of a text.

    Example: 
     <p style="font-style: italic">This text has italics.</p>
     
    This text has italics.

  4. font-family

    5. this property is used to change the font of a text.

    Example: 
     <p style="font-family: serif">This text has serifs.</p>
     
    This text has serifs.

  5. font-weight

    6. this property is used to add weight to texts.

    Example: 
     <p style="font-weight: 900">This text is extra bold.</p>
     
    This text is extra bold.

  6. text-align

    7. this property is used to change the position of the text. Either from left, center or right positions.

    Example: 
     <p style="text-align: right">This text is at the right.</p>
     
    This text is at the right.

  7. background-color

    8. this property is used to change the website's background colour.

    Example: 
     <body style="background-color blue: center">
     


  8. border

    9. this property is used to create borders which can be used by itself or added with texts.

    Example: 
     <p style="border: 3px; border-style: solid">This is in a border.</p>
     
     

    This is in a border.
